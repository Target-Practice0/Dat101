1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. you should use constants for values that do not change throughout the program, and variables for values that may change. Not all values can be constants, as some need to be updated or modified during program execution.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parentheses affect the order of operations in an expression. They ensure that the operations within the parentheses are performed first, which can change the final result of the expression.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. We can limit floating-point numbers to a given number of decimals using methods like toFixed() in JavaScript, which formats a number using fixed-point notation.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. To compare two floating-point values for equality, we can check if the absolute difference between them is smaller than a small threshold value (epsilon). For example, if the difference between 12.199999 and 12.2 is less than epsilon, we can consider them equal.

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators return boolean values: TRUE or FALSE. We can check their values directly in conditional statements or by logging them to the console.

  6. How does the Math.random() function work, and what can it be used for?

    7. The Math.random() function generates a pseudo-random floating-point number between 0 (inclusive) and 1 (exclusive). It can be used for various purposes, such as simulating randomness in games, generating random IDs, or selecting random items from a list.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. The modulus operator (%) returns the remainder of a division operation between two numbers. It is useful in various scenarios, such as determining if a number is even or odd, cycling through a set of values, or constraining values within a specific range.
